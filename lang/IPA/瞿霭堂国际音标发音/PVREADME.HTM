关于 PowerPoint Viewer

Microsoft Office PowerPoint Viewer 使您得以查看在 Microsoft Office PowerPoint 2000、PowerPoint XP、PowerPoint 2003 和 PowerPoint 2007 中创建的演示文稿。PowerPoint Viewer 还支持打开用密码保护的 PowerPoint 演示文稿。

注意：要查看在 PowerPoint 2007 中创建的演示文稿，需要安装针对 2007 Office Word、Excel 和 PowerPoint 文件格式的 Microsoft Office 兼容包。有关详细信息，请访问 http://office.microsoft.com/zh-cn。

系统要求

  • 具有 200 MHz 或更高频率处理器的 PC
  • Microsoft Internet Explorer 5 或更高版本
  • 支持的操作系统：Microsoft Windows 2000 Service Pack 4、Windows XP Service Pack 1、Windows Server 2003 或 Windows Vista。
  • 随机存取内存 (RAM)：
    • 对于 Windows 2000 Service Pack 4：64 MB 的 RAM
    • 对于 Windows XP Service Pack 1 和 Windows Server 2003：128 MB 的 RAM
    • 对于 Windows Vista：512 MB 的 RAM
  • 5 MB 的可用硬盘空间
  • Super VGA (800 x 600) 或更高分辨率的显示器

多语言问题

PowerPoint Viewer 不支持 Microsoft Office 校对工具。

PowerPoint Viewer 不支持在 PowerPoint 97 中创建的双语演示文稿。

使用 PowerPoint Viewer

PowerPoint Viewer 不能正确显示图表动画。图表将完全显示，而不显示动画。

PowerPoint Viewer 不支持打开信息权限管理 (IRM) 演示文稿。

PowerPoint Viewer 不支持运行宏和程序，也不支持打开链接式或嵌入式对象。

PowerPoint Viewer 不支持显示在演示文稿中链接的如下图形格式：.eps、.pct、.emz、.cdr、.cgm、.wpg 和 .wmz。
