shell out
付出

古代還沒有法定貨幣時，人們常以貝殼當貨幣使用，所以中文的「財」、「貨」等字中也都有「貝」。shell out這個動詞片語字面意思是「出自貝殼」，引申為「支付」，尤其是 指「付出大筆金錢」。

[The ministries] that shell out the funds run the projects, so if a governor thinks his prefecture doesn't need a proposed agricultural road, he can't redirect the money to build a nursing home.(出自：time，Apr. 24, 2000, p. 20 )
支付資金的〔日本政府部會〕也負責執行這些計畫，所以假如某一位知事認爲他管轄的縣市不需再開一條提議中的農業道路，他也不能更改用途而興建療養院。
