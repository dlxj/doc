can 还是 be able to？

can 还是 be able to？

．实用程度：90%
老师今天想跟各位同学谈谈can与be able to的关系。大部分的同学都知道can与be able to几乎是可以画上等号，可是在下面的情形，只有其中一个用法是正确的哟。中文是：拚了三天三夜，「我总算完成那个不可能的任务。」英文呢？

．中翻英：我总算完成那个不可能的任务。下列哪句英文才正确？
　(A) I could complete the impossible mission.
　(B) I was able to complete the impossible mission.

．文法小提示
can 与 be able to 基本上可以通用，但在表达完成一项艰巨的任务或事件时，只能用be able to。

．正确的答案是
　(A) I could complete the impossible mission.
　(B) I was able to complete the impossible mission.

．正解
　(B)没错，正解就是(B)。在众多的用法中，can 与 be able to 几乎是可以通用，唯独在表达完成一项艰巨的任务或事件时，只能用be able to。

　因为 can 的语气跟 be able to 比起来，弱了很多，指的是一般的能力，而be able to指的是要费尽千辛万苦而达成的能力，譬如说，弹弹钢琴可以说是can，但若是指能够弹古典钢琴名曲，那么 be able to 的用法就较佳了。

　「Can you play the piano？」与「Are you able to play Beethoven？」之间的差别，t了吗？
